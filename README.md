# Random Jokes Viewer

## About

This project is a modern jokes viewer application built using React and Axios. The application fetches random jokes from a public API and displays them inside an interactive card-based interface with a modern doodle-style UI.

Users can generate a new random joke dynamically using the refresh button.

---

## Features

- Fetch random jokes from API
- Display one joke at a time in a modern card layout
- Interactive button to load a new joke
- Custom loading animation
- Error handling for failed API requests
- Responsive and clean UI
- Modern doodle-inspired design

---

## Tech Stack

- React
- Vite
- Axios
- JavaScript (ES6+)
- CSS (custom styling)
